Quando um processo é executado, o é carregado com o endereço físico onde seu programa começa na e o registrador limite é carregado com o comprimento do programa. Toda vez que um processo referencia a memória, seja para buscar uma ou ler ou escrever uma palavra de dados, o hardware da CPU automaticamente adiciona o valor base ao endereço gerado pelo processo antes de enviá-lo para o barramento de memória. Registrador base e o são protegidos de tal maneira que apenas o sistema operacional pode modificá-los.
